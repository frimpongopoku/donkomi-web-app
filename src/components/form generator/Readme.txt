If you lift this form generator folder directly out of this repo, you would have to install 
the following to make the components work properly. 

Install 
------- 
1. Emotion/Css , Emotion/styled, Emotion/React : https://emotion.sh/docs/install
2. React-image-crop : https://www.npmjs.com/package/react-image-crop
3. @fortawesome/react-fontawesome : https://www.npmjs.com/package/@fortawesome/react-fontawesome 
4. @fortawesome/free-solid-svg-icons : https://www.npmjs.com/package/@fortawesome/free-solid-svg-icons

Install with the code below
npm install  --save @emotion/styled @emotion/react @emotion/css react-image-crop  @fortawesome/react-fontawesome  @fortawesome/free-solid-svg-icons
